# Workflow Expansion Backlog

이 문서는 1차 지식DB 확장 이후 이어서 추가할 세부분류와 워크플로우 후보를 정리한다.
`docs/taxonomy-expansion-candidates.md`가 중분류 후보 풀이라면, 이 문서는 실제 체인으로
전환하기 쉬운 작업 단위 후보 풀이다.

## Selection Rules

- 한 PR에는 한 대분류 또는 한 중분류의 후보만 추가한다.
- 새 체인은 `docs/workflow-chain-template.md`의 구조를 따른다.
- 체인마다 5~8개 단계, 단계별 이론 링크, 검색어, 실행 도구를 갖춘다.
- 먼저 사용자가 바로 실행할 수 있는 탐색/검증/정리 워크플로우를 고른다.
- 단순 자료 목록이 아니라 `정의 -> 근거 확인 -> 비교/분석 -> 실행/정리` 흐름을 우선한다.

## Priority 1. Breadth Balance

아래 후보는 각 대분류가 한 개 체인만 갖는 상태에서 두 번째 체인을 추가할 때 우선한다.

### 사회과학

세부분류 후보:
- 사회학·공동체
- 정치·정책학
- 커뮤니케이션 연구

워크플로우 후보:
- `지역사회 이슈 분석 플로우`
  - 문제 범위 정의
  - 인구·지역 통계 확인
  - 이해관계자 지도화
  - 유사 사례 탐색
  - 정책/제도 쟁점 정리
  - 실행 메모 작성
- `여론·미디어 반응 분석 플로우`
  - 이슈 키워드 정의
  - 뉴스/소셜 데이터 수집
  - 프레임과 논점 분류
  - 여론 변화 확인
  - 위험 신호와 기회 정리

### 자연과학

세부분류 후보:
- 물리·화학
- 생명과학
- 지구·우주과학

워크플로우 후보:
- `기초 과학 개념 검증 플로우`
  - 개념 정의 확인
  - 핵심 법칙/원리 탐색
  - 실험 또는 관측 사례 확인
  - 수식/모델 의미 정리
  - 학습 자료와 시각 자료 연결
- `생명과학 논문 탐색 플로우`
  - 생명 현상 키워드 정의
  - 리뷰 논문 탐색
  - 주요 실험 방법 확인
  - 데이터베이스 교차 확인
  - 연구 공백과 후속 질문 정리

### 의학·보건

세부분류 후보:
- 임상의학
- 정신건강
- 영양·운동건강

워크플로우 후보:
- `생활습관 건강 계획 플로우`
  - 목표와 제한 조건 정의
  - 신뢰 가능한 권고 확인
  - 식이/운동/수면 지표 정리
  - 추적 도구 선택
  - 의료 상담 질문 정리
- `정신건강 정보 탐색 플로우`
  - 증상과 상황 기록
  - 신뢰 출처에서 개념 확인
  - 자가진단 도구의 한계 확인
  - 상담/진료 접근 경로 정리
  - 위기 대응 자원 확인

### 법·정책

세부분류 후보:
- 민사·형사 법체계
- 규제·컴플라이언스
- 행정·공공정책

워크플로우 후보:
- `정책 제안 검토 플로우`
  - 정책 문제 정의
  - 법적 근거와 담당 기관 확인
  - 이해관계자와 영향 범위 정리
  - 국내외 유사 정책 비교
  - 실행 비용과 리스크 정리
- `규제 리스크 체크 플로우`
  - 서비스/제품 범위 정의
  - 적용 법령 후보 확인
  - 개인정보/소비자/광고 쟁점 점검
  - 규제기관 자료 확인
  - 검토 메모와 후속 질문 정리

### 농학·생명산업

세부분류 후보:
- 축산·식품시스템
- 바이오산업
- 산림·수산자원

워크플로우 후보:
- `식품 안전 정보 검증 플로우`
  - 식품 또는 원료 정의
  - 위해 요인과 기준 확인
  - 인증/표시 제도 확인
  - 유통과 보관 조건 검토
  - 소비자 안내 문구 정리
- `바이오산업 기술 탐색 플로우`
  - 기술 영역 정의
  - 논문과 특허 탐색
  - 적용 산업과 시장 확인
  - 규제와 윤리 쟁점 정리
  - 실험/사업화 단계 구분

## Priority 2. Existing Strong Categories

이미 여러 체인을 가진 대분류는 깊이를 보강한다.

### 공학·기술

워크플로우 후보:
- `보안 위협 모델링 플로우`
- `로보틱스 프로토타입 플로우`
- `제조 품질 개선 플로우`

핵심 단계 패턴:
- 자산/대상 정의
- 위험 또는 요구사항 도출
- 도구/데이터 수집
- 분석 모델 적용
- 결과 검토
- 문서화와 후속 조치

### 경영·경제

워크플로우 후보:
- `재무제표 기초 분석 플로우`
- `마케팅 채널 실험 플로우`
- `조직 운영 개선 플로우`

핵심 단계 패턴:
- 목표와 지표 정의
- 기준 데이터 수집
- 비교 기준 설정
- 실행안 설계
- 결과 측정
- 의사결정 메모 작성
