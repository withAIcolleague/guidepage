# Taxonomy Expansion Candidates

Guidepage의 지식DB 확장은 대백과사전형 대분류를 유지하면서, 각 대분류 아래에
대학교 학부 수준의 중분류를 먼저 세운 뒤 세부분류와 워크플로우 체인을 채운다.
이 문서는 런타임 데이터에 한꺼번에 넣을 목록이 아니라, 단계별 확장 PR의 후보
풀이다.

## Expansion Principles

1. 대분류는 사용자가 처음 보는 세계관 지도 역할을 한다.
2. 중분류는 학부/전공 수준의 개념 묶음이어야 한다.
3. 세부분류는 실제 워크플로우 체인 하나와 대응될 수 있을 만큼 구체적이어야 한다.
4. 한 PR에서 모든 후보를 추가하지 않고, 빈 대분류부터 최소 1개 워크플로우를 채운다.
5. 새 체인은 `docs/workflow-chain-template.md`와 `npm.cmd run validate:data`를 통과해야 한다.

## Priority

1. 빈 대분류에 첫 중분류/세부분류/체인을 채워 메인 카드의 준비 중 상태를 줄인다.
2. 이미 체인이 있는 대분류는 중분류 균형을 맞춘 뒤 깊이를 추가한다.
3. 도구, 이론 링크, 검색어가 명확한 실사용 워크플로우를 우선한다.
4. 단순 링크 모음이 아니라 정의, 학습, 실행이 이어지는 흐름을 우선한다.

## Candidate Sections

### 인문학

현재 섹션:
- 역사·예술사

확장 후보:
- 철학·사상사: 존재, 인식, 윤리, 정치사상, 미학의 개념 흐름
- 언어·문학: 언어 구조, 텍스트 해석, 장르, 서사와 비평
- 종교·신화: 종교 전통, 의례, 상징, 신화 체계
- 문화연구: 대중문화, 정체성, 젠더, 지역성과 문화 권력

### 사회과학

현재 섹션:
- 사회·제도 연구

확장 후보:
- 심리·행동과학: 개인 행동, 인지, 감정, 동기와 의사결정
- 사회학·공동체: 계층, 네트워크, 가족, 도시와 집단 구조
- 정치·정책학: 권력, 국가, 정책 형성, 거버넌스
- 커뮤니케이션 연구: 여론, 미디어 효과, 설득, 공론장

### 자연과학

현재 섹션:
- 물질·생명·지구

확장 후보:
- 물리·화학: 물질, 에너지, 반응, 구조와 실험 방법
- 생명과학: 세포, 유전, 생태, 진화와 생명 시스템
- 지구·우주과학: 지질, 기후, 해양, 천문과 우주 구조
- 환경과학: 생태계, 자원, 오염, 지속가능성 분석

### 수학·논리

현재 섹션:
- 계산·모델링

확장 후보:
- 순수수학: 대수, 해석, 기하, 위상과 증명 체계
- 통계·확률: 데이터 요약, 추정, 예측, 불확실성 판단
- 논리·형식체계: 명제, 술어, 증명, 계산 가능성
- 최적화·의사결정: 운영연구, 제약, 목적함수, 선택 모델

### 공학·기술

## First PR Candidates

아래 후보는 빈 대분류를 줄이는 데 우선순위를 둔다.

1. 사회과학: `심리·행동과학` 아래 `개인 의사결정 분석 플로우`
2. 자연과학: `환경과학` 아래 `기후·환경 데이터 탐색 플로우`
3. 의학·보건: `공중보건` 아래 `건강 정보 검증 플로우`
4. 법·정책: `규제·컴플라이언스` 아래 `서비스 약관·정책 검토 플로우`
5. 농학·생명산업: `작물·토양과학` 아래 `작물 재배 정보 탐색 플로우`

각 후보는 실제 체인 추가 전에 다음을 별도로 작성한다.

- 체인 목적
- 5~8개 단계
- 단계별 이론 링크
- 단계별 실행 도구 1~3개
- 검색어
- iframe 차단 가능성이 높은 도메인의 새 탭 CTA 여부
